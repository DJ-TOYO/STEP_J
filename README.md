# STEP_J
STEP_K (タグエディタ) の派生版

Kobarin 様作成のSTEP_Kを改造したものです。
GNU ライセンスに基づき公開いたします。 

Kobarin 様STEP_Kとの相違点
・セルが途切れたら加算する連番を実装
・高DPI環境向けにツールバーと大アイコンを追加
・既定のフォントをMeiryo UIに
となっています。フォントを変更した影響でWindows XPでは動作しないと思われます。
