■メニュー：ファイル

  • リストをクリア
    現在読み込まれているファイルの一覧をクリアします
  • フォルダを開く
    ユーザーが指定したフォルダからＭＰ３ファイルを検索し、リストに読み込みます
  • タグ情報を更新
    タグ情報が編集されたファイルのタグ情報をファイルに書き出します
  • リスト出力
    現在のファイルの一覧をユーザー指定の書式ファイルでリストを出力します
    書式ファイルについての詳細は [オプション]-[上級者向け]-[リスト出力] をご覧下さい。
  • 終了
    SuperTagEditor を終了します
    保存されていないファイルがある場合には、確認メッセージが表示されます
