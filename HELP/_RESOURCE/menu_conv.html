■メニュー：変換

  • [デフォルト変換]-[ファイル名 => トラック名]
    ファイル名の拡張子を取り除いた部分を トラック名 に変換します
    ※トラック名が空で無い場合は無視される
  • [デフォルト変換]-[トラック名 => ファイル名]
    トラック名 に拡張子を付けた名前を ファイル名 に変換します
    ※トラック名 が空の場合は無視される
  • [ユーザー指定変換]-[タグ情報 => ファイル名]
    ユーザーが指定した書式にしたがって、タグ情報からファイル名に変換します
  • [ユーザー指定変換]-[ファイル名 => タグ情報]
    ユーザーが指定した書式にしたがって、ファイル名からタグ情報に変換します
  • [拡張書式変換]
    ユーザーが指定した書式にしたがって、タグ情報を変換します
    ※連番機能対応、任意(範囲選択)のセルへの書き込み対応、任意文字列設定対応など
  • 大文字に変換
    選択されているセルの内容を大文字に変換します
  • 小文字に変換
    選択されているセルの内容を小文字に変換します
  • 半角に変換
    選択されているセルの内容を半角に変換します
  • 全角かなに変換
    選択されているセルの内容を全角かなに変換します
  • 全角カナに変換
    選択されているセルの内容を全角カナに変換します
  • 標準 MP3 形式に変換
    選択されているファイルを全て標準 MP3 形式に変換します
    ※編集中のファイルは変換できません
  • RIFF MP3 形式に変換
    選択されているファイルを全て RIFF MP3 形式に変換します
    ※編集中のファイルは変換できません
  • SIフィールドからID3tagに変換
    RIFF MP3ファイルのSIフィールドからID3tagに変換します
